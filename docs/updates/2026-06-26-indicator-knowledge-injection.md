# 指标知识注入

## 背景

企业问题中的“收入”“销售成本”“毛利”“期间费用”“利润”等词不是简单字段名，而是具有业务定义、公式、依赖关系和过滤条件的指标。系统需要显式管理这些指标口径。

## 本次变更

- 新增指标知识库，保存指标定义、别名、公式、数据来源、依赖指标和强制过滤条件。
- 新增指标知识模块，支持从用户问题中识别指标并生成 Prompt 知识块。
- 主查询流程自动注入相关指标知识，并在结果 metadata 中返回识别到的指标。
- 支持依赖指标补充，例如利润会带出毛利和期间费用，毛利会带出收入和销售成本。

## 关键文件

- `indicators.json`
- `indicator_knowledge.py`
- `prompt_builder.py`
- `main.py`

## 验证结果

- 指标识别可识别“利润”和“毛利”等业务词。
- 指标知识块可注入 Prompt。
- 查询“按产品线统计毛利”时，系统可返回识别指标 metadata，并生成包含收入、成本、汇率和订单过滤的 SQL。

## 后续衔接

指标知识将为后续更完整的指标检索、RAG 和复杂分析任务提供基础，也能减少模型对业务口径的自由猜测。
