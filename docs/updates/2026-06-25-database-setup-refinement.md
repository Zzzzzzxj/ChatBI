# 数据库环境完善

## 背景

初始数据环境建立后，需要让数据库脚本更便于重复执行、验证和重置，保证本地开发过程中可以快速恢复稳定数据状态。

## 本次变更

- 完善数据初始化与重置流程。
- 补充数据库验证查询，覆盖订单、收入、区域、产品线、毛利和费用等核心检查点。
- 增加从数据库读取结构并生成 Schema 文本的工具脚本。
- 更新数据环境说明，明确执行顺序和验证目标。

## 关键文件

- `sql/04_reset_data.sql`
- `scripts/schema_generator.py`
- `docs/database_setup.md`

## 验证结果

- 数据库可重复创建和重置。
- 验证查询可以确认核心表数据完整性。
- Schema 生成脚本可连接 MySQL 并输出 Prompt 所需结构描述。

## 后续衔接

后续查询链路可以直接复用稳定的数据环境，并把自动生成的 Schema 文本用于 Prompt 构造或调试。
