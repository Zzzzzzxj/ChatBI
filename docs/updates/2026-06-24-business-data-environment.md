# 业务数据环境建设

## 背景

ChatBI 需要一套能覆盖企业经营分析问题的业务数据环境。数据模型需要既能支持基础查询，也能暴露真实场景中的收入、成本、汇率、费用和维度关联问题。

## 本次变更

- 引入制造业和新能源业务域的核心数据模型。
- 建立客户、产品、销售订单、汇率、费用五类表。
- 提供初始化数据和基础验证查询，为 Text2SQL 生成结果提供可执行环境。
- 增加数据库结构说明，方便后续 Prompt 和 Schema 生成复用。

## 关键文件

- `sql/01_schema.sql`
- `sql/02_seed_data.sql`
- `sql/03_validation_queries.sql`
- `docs/database_setup.md`

## 验证结果

- 核心表结构可创建。
- 模拟数据覆盖订单状态、产品线、客户区域、币种汇率和费用字段。
- 验证查询可检查行数、收入聚合、汇率折算、毛利计算和费用汇总。

## 后续衔接

后续 Text2SQL 能力应基于这套数据环境生成可执行 SQL，并逐步暴露字段选择、Join 路径和业务口径问题。
